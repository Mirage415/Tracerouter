# Tracerouter 



https://github.com/Mirage415/Tracerouter



type and run the command included in ```bash ``` in your shell.



If you have downloaded the Tracerouter from zip
```bash
cd your_filepath/Tracerouter
```



if not, clone the respository
```bash
git clone https://github.com/your-username/Tracerouter.git
cd Tracerouter
```



install dependencies
```bash
npm install
```



run the Application
```bash
sudo npm start
```



if something went wrong, please read README.md for further information.
